# Анализ данных интернет‑магазина

Проект для анализа продаж, клиентов и товаров с использованием SQL.

## Структура базы данных

* **Customers** — клиенты;
* **Products** — товары;
* **Orders** — заказы;
* **Order_Items** — позиции заказов;
* **Suppliers** — поставщики.

## Как запустить

1. Установите MySQL Server (или используйте облачную БД).
2. Подключитесь к БД через VS Code (используйте SQL Tools) или другой клиент.
3. Выполните `schema.sql` для создания таблиц и связей.
4. Загрузите тестовые данные из `sample_data.sql`.
5. Запустите запросы из `queries.sql` для анализа.
